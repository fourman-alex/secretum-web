Secretum Web

Deterministic passphrase from a FIDO2 hardware key — fully client-side

How to use

  1. Click "Create resident key" to set up a FIDO2 key on your device (one-time setup)
  2. Enter any passphrase identifier (e.g., "github.com", "email") in the Entry field
  3. Click "Derive passphrase" to generate a unique passphrase
  4. Use the same device and identifier to always get the same passphrase
WebAuthn requires HTTPS or localhost. Serve this page over a secure origin.

Setup

Create a resident key on your FIDO2 hardware device. Do this once per device.
